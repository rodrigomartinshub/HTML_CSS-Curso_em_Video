Principais Formatações

Negrito/Destaque

Nesta frase temos uma palavra em negrito. (<b></b>) (forma)

Nesta outra temos uma palavra em destaque. (<strong></strong>) (semantica)

Nesta frase temos um termo em itálico. (forma)

Nesta aqui temos um termo em ênfase usando a tag <em></em>. (semantica)

Texto Marcado

Podemos também criar um texto marcado usando a tag <mark></mark>.

Outro texto marcado no final.

Texto Grande e Texto Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído (tag <del></del>) para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido (tag <ins></ins>) para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u></u>. (semantica)

Texto Sobrescrito e Subscrito

Para inserir coisas do tipo x20+3 inserimos a tag <sup></sup>.

Para inserir coisas do tipo H2O inserimos a tag <sub></sub>.
